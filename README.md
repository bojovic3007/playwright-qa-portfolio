# Playwright QA Automation Portfolio

## 🧪 Tech Stack
- Playwright
- TypeScript
- Node.js
- GitHub Actions
- Page Object Model (POM)

## 📁 Project Structure
- pages/           -> Page Object Model (POM)
- tests/ui/        -> UI tests
- tests/api/       -> API tests
- test-data/       -> test data
- utils/           -> helper functions
- fixtures/        -> test setup

## ▶️ How to run tests

Install dependencies:
    npm install

Run all tests:
    npx playwright test

Run tests in UI mode:
    npx playwright test --ui

Generate report:
    npx playwright show-report

## 📊 CI / GitHub Actions

This project is using GitHub Actions for automatic run for all tests on evry push. Status can be seen in Actions tab on GitHub.
